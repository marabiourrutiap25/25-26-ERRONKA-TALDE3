# 25-26-ERRONKA-TALDE2

<h2>1️⃣ XAMPP instalatu.</h2>
<p>https://www.apachefriends.org/</p><br>

<h2>2️⃣ Errepositorio XAMPP-en 'htdocs' karpetan klonatu:</h2>
<p>git clone https://github.com/marabiourrutiap25/25-26-ERRONKA-TALDE3.git</p><br>

<h2>3️⃣ Apache eta MySQL-ren konfigurazio basikoak egin.</h2>
<p>Horien artean, hauek egongo dira.<br>
Apache → 80 / 443 portuak.<br>
MySQL → 3306 portua (edo beste bat, gero konfiguratu ahal izango da).</p><br>

<h2>4️⃣ XAMPP-ekin 'Erronka1' datubasea sortu.</h2>
<p>Jadanik sartuta, hau idatzi behar da: 'create database Erronka1;', eta datu basean sartu behar gara: 'use Erronka1;'.<br>
Ondoren, '25-26-ERRONKA-TALDE3\backend\scripts'-n dauden scriptak kopiatu eta itsatsi behar dira, 1.go 'Taulak.sql' eta jarraian 'Datuak.sql'.</p><br>



<h2>5️⃣ Datu basearen kredentzialak hurrengo fitxategian konfiguratu:</h2>
<p>25-26-ERRONKA-TALDE3\backend\config.php</p><br>

<h2>6️⃣ XAMPP-en Apache eta MySQL abiarazi eta XAMPP-en karpeta bilatu:</h2>
<p>http://localhost/25-26-ERRONKA-TALDE3</p><br>

<h2>7️⃣ Helbideekin arazo bat izatekotan, hemen alda daitezke:</h2>
<p>25-26-ERRONKA-TALDE3/frontend/config.js</p><br>
